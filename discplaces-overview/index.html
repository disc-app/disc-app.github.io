DiscPlaces Introduction

DiscPlaces is an app for hyperlocal search. It’s extremely simple: just a map overlaid with thumbnails of posts, and a searchbar on top. Search is contextual, i.e., search suggestions are based on what’s close to what you are searching for. Some additional features include

  • Showing posts for a single level of a building.
  • The ability to of beacons for extremely accurate positioning, as well as alerts.
  • Selecting places to search in using place search with suggestions.

Content for DiscPlaces comes from the companion DiscApp, which allows content creators to generate and edit rich content very easily.

DiscPlaces will become an open source app - it can be used standalone, or embedded into existing apps. For example, a retailer can embed DiscPlaces into their app, allowing them to give their customers the ability to find items in their store. By using an API key we provide, the embedded DiscPlaces capability can be restricted to just the content created by a particular user or set of users.
